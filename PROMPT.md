# Prompts

CLAUDE.md is loaded automatically — no need to reference it or ask to load rule files.
Use slash commands where possible; fall back to text prompts when needed.

---

## Full integration

```
/integrate
```

Or as text:
```
Perform a full integration. Start with the audit and report sections before writing any code.
```

---

## Single section

```
/section hero
```

Or as text:
```
Convert the hero section.
```

---

## Continue after a stop

```
Continue integration. Pick up from where we left off.
```

---

## Audit JSON

```
/audit-json
```

Or as text:
```
Audit the ACF JSON file and report all issues.
```
